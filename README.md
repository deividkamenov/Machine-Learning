## Machine Learning – Homework Collection

Collection of homework assignments for an **Intelligent Systems / AI** university course.

Each subfolder (`HW1`, `HW2`, `HW3`, `HW4`, `HW5`, `HW7`, `HW8`) contains a separate task – such as search algorithms, knowledge representation, basic machine learning, planning, etc. (depending on the exact assignment specification).

### Structure

- `HW1/` – Homework 1
- `HW2/` – Homework 2
- `HW3/` – Homework 3
- `HW4/` – Homework 4
- `HW5/` – Homework 5
- `HW7/` – Homework 7
- `HW8/` – Homework 8

Inside each folder you will typically find:

- Source code (Python / other language depending on the task)
- Example input / output data
- Short write‑ups or PDF statements (if provided by the course)

### How to run

- Check the README / comments or task statement inside each `HW*` folder for language and run instructions.
- If the homework is in Python, usually:

```bash
python main.py
```

or another entry script, with any required input files.
